## 八、后续演化可能性

### 情景 A — 基准：MOU 执行爬坡 (概率 ~55%)

- **触发条件**：Hormuz 过境数稳步回升至 50+ /日；IAEA 与 Iran 在 6/25-26 就核查 timing 达成口头共识；黎南 truce 不破；Rubio Bahrain 之行无新风波。
- **市场反应**：Brent **$71–77** / WTI **$68–73** 区间整理；S&P 500 高位震荡 ±1-2%；Nasdaq 100 继续 -2 至 -4% 的资金分流；DXY 102-103；10Y 4.05-4.15%。
- **板块**：能源股进一步 -3 至 -5%，航运 / 航空 / 旅游 / 银行 / 中小盘延续；半导体 / AI 大盘股继续被资金减仓 -2 至 -5%。
- **风险**：MOU 文本爬坡过程中任一公开错位都可能引发 ±$3-5/桶级别脉冲；散户 AI 多头 vs 机构对冲基金的持仓背离已到极端水位，反弹也可能集中爆发。

### 情景 B — 上行（升级）：MOU 文本翻车 / 二线升级 (概率 ~25%)

- **触发条件**：Iran 第二次单方面关闭海峡（类比 6/20 事件）；以军升级对真主党据点 / 南黎攻击；Hezbollah 报复；6/24 类型核查口角进一步升级至公开撕毁 MOU；或 Iran 对商船实施"许可文件审查"实质中断海峡通行。
- **市场反应**：Brent 上探 **$82–95** / WTI **$78–92**；S&P 500 -3 至 -6%（能源板块对冲掉部分跌幅）；VIX 跳至 22-26；黄金回探 $2,500+；美 10Y 收益率冲高后回落（避险 bid）。
- **板块**：能源股 +5 至 +12%；国防 +3 至 +6%；航运股先涨（费率重溢价）后跌（货量再次归零）；旅游 / 航空 / 邮轮 / 消费集体 -4 至 -7%；AI 抛售或暂停（资金回流避险）。
- **风险**：尾部包含 Iran 攻击沙特 Aramco 油田 → Brent 一日跳涨至 $110-120 极端情形；阿曼临时航道一旦被攻击 / 关闭，复航预期会瞬时反向。

### 情景 C — 下行（缓和）：完整复航 + 保费回落 (概率 ~20%)

- **触发条件**：过境数恢复至 80+ /日（接近战前 86%+）；Lloyd's 公布 VLCC 战险费率分阶段回落机制；UAE / Saudi 持续超产；IAEA 与 Iran 就核查全流程达成正式协议；中俄在 UN 不再设阻。
- **市场反应**：Brent 下探 **$66–71** / WTI **$63–68**；S&P 500 +2 至 +4%（非 AI 板块带动）；DXY 进一步走软至 101.5；金价跌至 $2,370 一线。
- **板块**：能源股 -8 至 -14%；航运 / 旅游 / 航空 / 银行 / 中小盘集体 +3 至 +7%；半导体 / AI 大盘股在"流动性回流" trade 下可能 short-cover +3 至 +5%；电动车 / 替代能源短线遭获利了结。
- **风险**：协议执行良好后通胀预期过快下行 → 美联储 9 月降息预期回升（CME FedWatch 概率从 ~55% 升至 ~80%）→ 进一步利好风险资产，但同时放大 2027 通胀反弹尾部风险。
